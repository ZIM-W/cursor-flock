Cursor Flock — Preview Build
Installation
1. Drag Cursor Flock.app into Applications.
2. Open Cursor Flock from Applications.
3. If macOS blocks the app because the developer cannot be verified:
   - Open System Settings.
   - Go to Privacy & Security.
   - Find the warning for Cursor Flock.
   - Click Open Anyway.
   - Confirm Open.
This preview build is distributed through GitHub Releases and is not notarized by Apple.
After launch, use the Cursor Flock icon in the macOS menu bar to enable, disable, and configure the flock.
To uninstall, quit Cursor Flock and move it from Applications to Trash.
